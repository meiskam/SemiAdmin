SemiAdmin v0.35h
================

Unreal Tournament GOTY edition (UT99) server plugin

This is Hobbit's "private" modification of SemiAdmin, released with permission from Hobbit

Permission nodes:
* B = ban
* G = change game
* K = kick
* L = set up ladder
* M = change maps
* O = manage bots
* S = summon
* U = can manage users
* W = warning
* Z = zerofrag, respawn, mute, unmute
* T = ??
* F = do full (all hob commands)
* A = fly, walk, ghost
* H = goto
* D = god, health
* C = slap
* E = boost
* V = in/visible
